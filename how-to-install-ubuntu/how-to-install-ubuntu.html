How to install Ubuntu Linux 16.04.5 LTS on Virtual Machine

  1. VMware Workstation 15 Pro를 실행한다

  2. "Create a New Virtual Machine"을 클릭한다.

  3. "Typical(recommanded)"를 선택한 후, "Next"버튼을 클릭한다.

  4. "Installer disc image file (iso):"를 선택한 후, "Browse..."버튼을 클릭하여 앞서 다운로드를 하여 두었던 "ubuntu-16.04.5-desktop-amd64.iso" 파일을 선택한다. 그리고 "Next" 버튼을 클릭한다.

  5. "Full name"란에 full name을 기입한다. "User name"란에 name을 기입한다. "Password", "Confirm"란에 비밀번호를 기입한다. 그리고 "Next" 버튼을 클릭한다.

  6. "Virtual machine name"란에 사용할 가상 머신 이름을 기입한다. "Browse..." 버튼을 클릭하여 "Location"란에 앞으로 가상머신 파일이 저장될 폴더를 지정한다. 그리고 "Next" 버튼을 클릭한다.

  7. "Maximum disk size (GB)" 란에 사용할 Hard Disk의 크기를 기입한다. 좋은 성능을 위하여 "Store virtual diak as a single file"을 선택한다. 그리고 "Next" 버튼을 클릭한다.

  8. "Customize Hardware..." 버튼을 클릭한다.

  9. "Memory | 2GB"를 클릭한다(VMware Workstation 15 Pro는 기본적으로 2GB로 설정한다). "2 GB"를 "8 GB"로 수정한다(현재 사용하고 있는 PC의 최대 메모리 사이즈를 넘어서 설정하면 않된다).

  10. "Processors | 1"를 클릭한다(VMware Workstation 15 Pro는 기본적으로 1로 설정되어 있다). "Number of processors"를 "1"에서 "4"로 수정한다(현재 사용하고 있는 PC의 최대 프로세서 갯수를 넘어서 설정하면 않된다).

  11. 하드 디스크의 사이즈, 메모리 사이즈, 프로세서의 갯수에 대한 설정을 포함하여 다른 모든 하드웨어 설정들은 나중에 언제든지 수정이 가능하다. "Close" 버튼을 클릭한다.

  12. "Power on this virtual machine after creation"의 체크박스를 체크하고 "Finish" 버튼을 클릭한다.

  13. 자동으로 Ubuntu가 설치되는 것을 확인할 수 있다. Ubuntu 설치가 끝날 때까지 커피 한잔의 여유를 즐기면서 기다린다.

  14. Ubuntu 설치가 끝나면 다음과 같이 비밀번호를 기입하라는 화면이 나타난다. 비밀번호 기입란에 마우스 커서를 놓고 더블클릭한다. 비밀번호를 입력한다. 키보드 "Enter" 키를 누른다.

  15. VMware Workstation 15 Pro의 top menu에 있는 "Enter full screen mode" 아이콘을 클릭한다.

  16. 다음과 같이 전체화면으로 Ubuntu가 보이게 된다.

  17. Ubuntu 로그인 후, 일정시간이 지나면 "Software Updater" 아이콘이 나타난다. "Software Updater" 아이콘을 클릭한다.

  18. "Install Now" 버튼을 클릭한다.

  19. 다음과 같은 화면이 나타나면 비밀번호를 입력하고 "Authenticate" 버튼을 클릭한다.
